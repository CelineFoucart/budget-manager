# budget-manager

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run electron:serve
```

### Build the app
```
npm run electron:build
```

En cas d'erreur :
````bash
$env:NODE_OPTIONS = "--openssl-legacy-provider"
``

### Lints and fixes files

```
npm run lint
```
